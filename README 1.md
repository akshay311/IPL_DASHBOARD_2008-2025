# IPL_DASHBOARD_2008-2025
Necessity
As a high-voltage form of entertainment,IPL data is vast -stats on players, teams, venues, seasons, and match outcomes. This data needs to be consolidated in an Cricket data analytics dashboard with a user-friendly interface to provide a better understanding for teams, analysts, and fans.

To address this requirement, we are pleased to come up with a comprehensive dashboard for  IPL stats. This Power BI dashboard provides easy access to an interactive exploration of key metrics for sports enthusiasts.

Usability
This IPL analytics dashboard provides key insights, including player profiles, head-to-head matchups, team rivalries, venue performance, and historical records. It helps evaluate consistency for fantasy picks, with custom filters for seasons, teams, players, and overs, making data-driven decisions easier.

Key Features
Batter & Bowler Profiles showing individual stats, strengths and performance trends.
Player vs Player head-to-head uncovering the match-ups between the batter and bowler.
Team vs Team head-to-head uncovering the performance and rivalries of Teams against each other.
Teams performance at specific venue and toss impact.
Performance, records and Title glories of teams across the history of the tournament displaying most successful teams.
A user-friendly tool that distributes head-to-head player performance, helping evaluate consistency and make data-driven fantasy picks for optimal matchups.
Filtering based on season, Teams, venues and Players & overs
